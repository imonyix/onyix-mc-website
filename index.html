How to enable the live map

The map requires the Dynmap plugin installed on your server:

  1. Download Dynmap from SpigotMC
  2. Put Dynmap.jar in plugins/ folder
  3. Set webserver-port to 25571 in configuration.txt
  4. Restart server and run dynmap fullrender
